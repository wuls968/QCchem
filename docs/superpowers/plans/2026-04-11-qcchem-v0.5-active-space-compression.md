> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Extend QCchem with stronger active-space workflows, compression/low-rank audit, NEVPT2 correction, and embedding/fragmentation skeletons while preserving the existing run/benchmark/study/scan artifact model.

**Architecture:** Keep `RunResult` as the atomic artifact and add four new chemistry-facing sections: richer reduction/active-space audit, compression audit, perturbative correction result, and embedding audit. Put chemistry-side transformation logic in `chem/`, schema in `core/`, config parsing in `io/`, orchestration in `workflow/runner.py`, and report rendering in `reporting/markdown.py`.

**Tech Stack:** Python, Qiskit, Qiskit Nature, Qiskit primitives V2, Qiskit Aer, PySCF, PyYAML, pytest.
